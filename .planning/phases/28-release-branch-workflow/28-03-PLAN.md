**1. Modify display_bundle_diff() to show full release notes:**

In `display_bundle_diff()` (around lines 415-433), replace the truncated release notes display. Currently:

```bash
    # Show release notes (up to 10 lines)
    if [[ -n "${RELEASE_NOTES:-}" ]]; then
        echo ""
        echo "Release notes:"
        local line_count=0
        while IFS= read -r line; do
            [[ -z "$line" && $line_count -eq 0 ]] && continue
            echo "  ${line}"
            line_count=$((line_count + 1))
            [[ $line_count -ge 10 ]] && break
        done <<< "${RELEASE_NOTES}"
        local total_lines
        total_lines=$(echo "${RELEASE_NOTES}" | wc -l)
        if [[ "$total_lines" -gt 10 ]]; then
            echo "  ... (${total_lines} lines total)"
        fi
        echo ""
        echo "Full changelog: https://github.com/botAGI/AGmind/releases/tag/${RELEASE_TAG}"
    fi
```

Replace with:

```bash
    # Show full release notes
    if [[ -n "${RELEASE_NOTES:-}" ]]; then
        echo ""
        echo -e "${BOLD}Release notes (${RELEASE_TAG}):${NC}"
        echo ""
        while IFS= read -r line; do
            echo "  ${line}"
        done <<< "${RELEASE_NOTES}"
        echo ""
        echo -e "  ${CYAN}https://github.com/botAGI/AGmind/releases/tag/${RELEASE_TAG}${NC}"
    fi
```

This removes the 10-line truncation and shows the full notes. The GitHub URL is still shown at the bottom for reference.
