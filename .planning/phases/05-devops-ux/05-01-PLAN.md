5. **Backup section** — call `check_backup_status` (reuse from health.sh)
6. **Credentials section** — show path `/opt/agmind/credentials.txt` only, NEVER content

Format: section headers using `echo -e "${BOLD}Section:${NC}"` consistent with health.sh.

`_status_as_json()` — must NOT call check_all() or other text-output functions (pitfall #6). Collect data directly:
1. Get service list from `get_service_list`, check each via `docker compose ps --format '{{.Status}}'`, count running/total
2. Build services details JSON object: `{"db":"running","redis":"running",...}`
3. Get GPU info from nvidia-smi / .agmind_gpu_profile (same logic as check_gpu_status but capture values, not print)
4. Get model list from `docker compose exec -T ollama ollama list` if LLM_PROVIDER=ollama
5. Build endpoints object from .env
6. Assemble full JSON matching this exact schema:
```json
{
  "status": "healthy|degraded|unhealthy",
  "timestamp": "ISO8601",
  "services": {
    "total": N,
    "running": M,
    "details": {"db": "running", "redis": "running"}
  },
  "gpu": {"type": "nvidia|amd|intel|none", "name": "...", "utilization": "..%"},
  "endpoints": {"webui": "http://...", "dify": "http://...:3000"},
  "backup": {"last": "2026-...", "age_hours": N, "status": "ok|warning|stale|none"}
```

Use printf/cat heredoc for JSON assembly. Escape double-quotes in values with `sed 's/"/\\"/g'`. Write to stdout only.

Overall status logic: if all services running → "healthy"; if some running → "degraded"; if none running → "unhealthy".

**cmd_doctor implementation:**

```bash
cmd_doctor() {
    local errors=0 warnings=0 checks=()
    local output_json=false
    while [[ $# -gt 0 ]]; do
        case "$1" in
            --json) output_json=true; shift ;;
            *) echo -e "${RED}Неизвестный флаг: $1${NC}" >&2; exit 1 ;;
        esac
    done

    _check() {
        local severity="$1" label="$2" message="${3:-}" fix="${4:-}"
        if [[ "$output_json" == "true" ]]; then
            local msg_escaped fix_escaped
            msg_escaped=$(echo "$message" | sed 's/"/\\"/g')
            fix_escaped=$(echo "$fix" | sed 's/"/\\"/g')
            checks+=("{\"severity\":\"${severity}\",\"label\":\"${label}\",\"message\":\"${msg_escaped}\",\"fix\":\"${fix_escaped}\"}")
        else
            case "$severity" in
                OK)   echo -e "  ${GREEN}[OK]${NC}   $label" ;;
                WARN) echo -e "  ${YELLOW}[WARN]${NC} $label — $message"
                      [[ -n "$fix" ]] && echo -e "         ${CYAN}-> $fix${NC}"
                      ;;
                FAIL) echo -e "  ${RED}[FAIL]${NC} $label — $message"
                      [[ -n "$fix" ]] && echo -e "         ${CYAN}-> $fix${NC}"
                      ;;
                SKIP) echo -e "  ${CYAN}[SKIP]${NC} $label — $message" ;;
            esac
        fi
        case "$severity" in
            WARN) warnings=$((warnings+1)) ;;
            FAIL) errors=$((errors+1)) ;;
        esac
    }
```
